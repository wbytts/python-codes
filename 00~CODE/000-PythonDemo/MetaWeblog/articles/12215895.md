# OpenCV 是什么

OpenCV是计算机视觉开源库，主要算法涉及图像处理和机器学习相关方法。

是 Intel 公司贡献出来的，俄罗斯工程师贡献大部分 C/C++ 代码

在多数图像处理相关的应用程序中被采用BSD许可，可以免费应用在商业和研究领域
